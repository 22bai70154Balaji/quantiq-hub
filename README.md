# Overview

Calculyx AI is a modern AI-powered financial intelligence platform that helps individuals make smarter financial decisions through interactive calculators, intelligent insights, professional reports, and financial planning tools.

The platform combines financial calculators with AI to provide users with actionable recommendations instead of just numerical outputs.

# Installation

Clone the repository

```bash
git clone https://github.com/yourusername/calculyx-ai.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production

```bash
npm run build
```
